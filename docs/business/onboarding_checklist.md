# AutoDiscovery Onboarding Checklist

Welcome! Follow these steps to get started with AutoDiscovery:

1. **Access the Application**
   - Obtain login credentials from your admin.
   - Log in at [app URL].

2. **Initial Setup**
   - Set your profile and notification preferences.
   - Connect required network segments (if applicable).

3. **Run Your First Discovery**
   - Go to the “Discover” tab.
   - Select a network or host group.
   - Click “Start Discovery.”
   - Monitor real-time progress in the dashboard.

4. **Review and Export Results**
   - Upon completion, review findings in the “Reports” tab.
   - Export data as CSV or PDF.

5. **Set Up Scheduled Discoveries** (Admins Only)
   - Navigate to “Schedules.”
   - Configure time, scope, and notifications.
   - Save and enable automated runs.

6. **Need Help?**
   - Access built-in help via the [?] icon.
   - Contact support at [support email].

You're all set!
