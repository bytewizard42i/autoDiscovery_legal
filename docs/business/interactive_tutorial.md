# Quickstart: Your First Network Discovery

Welcome to AutoDiscovery! Let’s walk through your first primary workflow:

1. **Login to AutoDiscovery.**
2. **Navigate to the Discover tab.**
3. **Select Your Network or Range.**
    - Enter subnet or select from the dropdown.
4. **Configure Scan Options.**
    - Optionally, set scan type and agents.
5. **Start Discovery.**
    - Click 'Start'.
    - Follow real-time progress.
6. **Review the Report.**
    - Go to 'Reports'.
    - Inspect details; filter or export.
7. **Schedule Automated Scans** (Optional, Admin)
    - Visit 'Schedules'.
    - Set time/date, targets, notifications.

Explore other features or consult the Help menu for advanced usage!
